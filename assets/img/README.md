# 일러스트 에셋

UI는 아래 파일명을 참조하며, 파일이 없으면 자동으로 이모지/그라데이션 폴백으로 동작합니다.

| 파일 | 용도 |
|---|---|
| `portrait_liubei.png` | 유비 초상화 (대화/영웅 선택) |
| `portrait_guanyu.png` | 관우 초상화 |
| `portrait_zhangfei.png` | 장비 초상화 |
| `portrait_zhaoyun.png` | 조운 초상화 |
| `portrait_zhugeliang.png` | 제갈량 초상화 |
| `portrait_caocao.png` | 조조 초상화 (대화) |
| `portrait_taoqian.png` | 도겸 초상화 (대화) |
| `title_bg.png` | 타이틀 화면 배경 일러스트 (16:9) |
| `map_bg.png` | 전장 지도/영웅 선택/결과 화면 배경 (16:9) |

9종 모두 Higgsfield 계정의 생성 기록(2026-06-12)에 동일한 스타일 공식으로 생성되어 있습니다.
다운로드 후 위 파일명으로 이 폴더에 넣으면 즉시 적용됩니다.
