# 삼국지 디펜스 · 유비전 (三國志 Defense)

킹덤 러쉬(Kingdom Rush)를 모티브로 한 삼국지 배경의 HTML5 타워디펜스 게임입니다.
유비를 주인공으로, 황건적의 난부터 한중 공방전까지 **10개의 역사적 전투**를 스테이지로 진행합니다.

## 실행 방법

별도 빌드 없이 정적 파일로 동작합니다.

```bash
# 저장소 루트에서
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

또는 `index.html`을 브라우저로 직접 열어도 됩니다.

## 게임 소개

### 스테이지 (유비의 일대기)

| # | 전투 | 연도 | 보스 |
|---|------|------|------|
| 1 | 황건적의 난 | 184 | 장각 |
| 2 | 호로관 전투 | 191 | 화웅 |
| 3 | 서주 구원전 | 194 | 조인 |
| 4 | 소패 공방전 | 198 | 여포 |
| 5 | 여남 전투 | 201 | 장합 |
| 6 | 박망파 전투 | 202 | 하후돈 |
| 7 | 장판파 전투 | 208 | 조순 |
| 8 | 적벽대전 | 208 | 채모 |
| 9 | 낙성 전투 | 213 | 장임 |
| 10 | 한중 공방전 | 219 | 하후연 |

각 스테이지는 전투 전후로 시나리오 대화가 진행되며, 스테이지를 클리어하면 다음 전장이 해금됩니다 (진행 상황은 localStorage에 저장).

### 타워 (삼국시대 병기)

- 🏹 **궁병탑** → 강궁병탑 → 제갈연노: 빠른 단일 공격
- ⚔️ **창병막사** → 극병막사 → 백이병막사: 길목에 창병 2명을 배치해 적의 진군을 차단
- 🪨 **투석기** → 대형투석기 → 벽력거: 느리지만 강력한 범위 피해
- 🔥 **화공탑** → 맹화유탑 → 업화탑: 방어구 무시 화염 피해 + 화상 지속 피해

각 타워는 3단계까지 업그레이드, 60% 환불 판매 가능.

### 영웅 레벨업

적을 처치하면 영웅이 경험치를 얻습니다 (영웅 근처 처치는 전액, 원거리 부대 처치는 30%).
레벨이 오르면 체력·공격력 +8%, 필살기 위력 +6%씩 강해지며 최대 Lv.10까지 성장합니다.
경험치는 영웅별로 저장되고, 패배해도 절반은 보존됩니다.

### 도전 과제 (스테이지별 3종)

- 🛡️ **완벽 방어**: 목숨을 하나도 잃지 않고 승리
- ⚔️ **불사 영웅**: 영웅이 한 번도 쓰러지지 않고 승리
- ⏱️ **속전속결**: 모든 공세를 조기 소집하여 승리

달성한 과제는 전장 지도의 스테이지 카드에 배지로 표시됩니다.

### 그래픽

외부 이미지 없이 코드로 픽셀아트를 절차 생성합니다 (`js/sprites.js`).
보병·기마·공성차 유닛(2프레임 보행 애니메이션, 방향 반전), 동양식 누각 타워(레벨별 외형 변화),
출발 군문(軍門)과 도착 요새, 테마별 지형(평원/숲/산악/강)과 들꽃·대나무·민가 장식,
파티클(폭발 파편·흙먼지·불씨)과 검격/낙뢰/화계 이펙트를 포함합니다.

### 사운드

Web Audio API로 효과음과 동양풍 BGM을 절차 생성합니다 (외부 에셋 없음).
리버브·컴프레서 체인 위에 가야금풍 플럭(디튠 2현), 비브라토 피리 리드, 큰북/소고/딱따기 장단으로
A/B 2부 구성의 펜타토닉 멜로디를 연주합니다. HUD의 🔊 버튼 또는 `M` 키로 음소거할 수 있습니다.

### 영웅과 필살기

스테이지마다 영웅 1명을 선택해 출전합니다. 영웅을 클릭한 뒤 지도를 클릭하면 이동하며, `Q`/스페이스 또는 우하단 버튼으로 필살기를 사용합니다.

- 👑 **유비** — 쌍고검 난무: 주변 광역 피해 + 기절
- 🟥 **관우** (2스테이지 해금) — 청룡언월참: 대범위 고화력 일격
- 🐍 **장비** (3스테이지 해금) — 장판교의 포효: 전장 전체 3초 기절
- ⚪ **조운** (5스테이지 해금) — 일기당천 돌격: 적진 관통 돌진
- 🪶 **제갈량** (6스테이지 해금) — 천화료원(화계): 적 밀집 지점에 불비

적군 보스 무장(여포, 하후돈, 하후연 등)도 고유 스킬을 사용합니다.

### 조작

- 🚩 깃발 부지 클릭 → 타워 건설 / 업그레이드 / 판매
- 웨이브 사이 간격은 **10초** — 배너 클릭으로 조기 시작하면 보너스 골드
- `⏩` 2배속, `⏸` 일시정지, 🔊 음소거 지원
- 남은 목숨에 따라 별 1~3개 획득 (18 이상: ⭐⭐⭐)
- 모바일/터치 화면 지원 (반응형 레이아웃, 넓은 터치 판정)

## 구조

```
index.html      # 마크업 및 화면(타이틀/지도/대화/영웅선택/결과)
css/style.css   # UI 스타일 (동양풍 테마, 모바일 반응형 포함)
js/audio.js     # 사운드 (Web Audio API 절차 생성 효과음/BGM)
js/sprites.js   # 픽셀아트 스프라이트/지형 절차 생성
js/data.js      # 타워·적·보스·영웅·스테이지·시나리오 데이터
js/game.js      # 게임 엔진 (루프, 전투, 렌더링, UI, 레벨업, 도전과제)
```
